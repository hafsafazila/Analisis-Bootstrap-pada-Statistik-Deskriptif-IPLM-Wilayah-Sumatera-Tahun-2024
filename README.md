# Analisis Bootstrap pada Statistik Deskriptif IPLM Wilayah Sumatera Tahun 2024

**Deskripsi Proyek Poster Komputasi Statistika**

**Pendahuluan**  
Proyek ini disusun oleh Kelompok 1 dalam mata kuliah Komputasi Statistika dengan fokus pada analisis perbandingan statistik deskriptif Indeks Pembangunan Literasi Masyarakat (IPLM) di wilayah Sumatera tahun 2024. Penelitian ini bertujuan mengevaluasi kestabilan estimasi statistik menggunakan metode bootstrap agar hasil analisis lebih akurat, stabil, dan reliabel meskipun data terbatas.

**Variabel Penelitian**  
Penelitian menggunakan variabel utama sebagai berikut:  
- Indeks Pembangunan Literasi Masyarakat (IPLM) sebagai indikator tingkat literasi masyarakat  
- Mean (rata-rata) sebagai ukuran pemusatan data  
- Median sebagai nilai tengah distribusi data  
- Standar Deviasi (SD) sebagai ukuran variasi data  
- Bootstrap Resampling sebagai metode estimasi berbasis pengambilan sampel ulang  

**Metode yang Digunakan**  
Analisis dilakukan menggunakan pendekatan kuantitatif berbasis data IPLM tahun 2024 dari 135 kabupaten/kota di Sumatera. Metode bootstrap diterapkan melalui proses preprocessing data, resampling dengan replacement, perbandingan statistik deskriptif sebelum dan sesudah bootstrap, serta visualisasi distribusi menggunakan RStudio.

**Hasil dan Insight**  
Hasil menunjukkan bahwa nilai mean, median, dan standar deviasi sebelum dan sesudah bootstrap berada pada kisaran yang hampir sama. Hal ini menandakan estimasi statistik yang stabil dan tidak sensitif terhadap variasi sampel. Distribusi data juga cenderung mendekati normal, sehingga bootstrap berhasil meningkatkan reliabilitas hasil analisis.

**Kesimpulan**  
Metode bootstrap terbukti efektif dalam menghasilkan estimasi statistik deskriptif yang lebih stabil, akurat, dan representatif pada data IPLM Sumatera 2024. Pendekatan ini memberikan dasar analisis yang lebih kuat untuk evaluasi pembangunan literasi masyarakat.

**Saran**  
Bootstrap disarankan untuk penelitian dengan keterbatasan data karena mampu meningkatkan stabilitas estimasi. Penelitian selanjutnya dapat menambahkan variabel sosial-ekonomi lain atau metode inferensial tambahan agar hasil analisis semakin komprehensif dan mendukung kebijakan pembangunan literasi yang lebih tepat sasaran.
